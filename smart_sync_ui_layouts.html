Спецификация Экранных Форм Smart Sync v1.0.1

Интерактивный эталон верстки терминала (жесткая сетка 90 символов, ANSI палитра)
Ниже представлены три ключевых состояния консольного интерфейса, спроектированные в точном соответствии с требованиями ТЗ v1.0.1. Пути приводятся относительно корня сетевой шары (сам корень отсекается), сжатие строк выполняется через центральный маркер ... ..., счетчики и спиннеры работают независимо для каждого запущенного потока.
1. Сценарий: Zero_Start (Первый запуск, базы знаний нет)
Критическая особенность: общий прогресс-бар полностью скрыт. На его месте выводится розовое информационное сообщение. Общий счетчик файлов равен Обработано / [сбор данных...].
